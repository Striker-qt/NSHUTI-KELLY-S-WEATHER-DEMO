# Weather App by NSHUTI KELLY

## How to run the app
1. Open `index.html` in any web browser (Chrome, Firefox, etc.). Double-click the file or drag to browser.
2. Get free API key:
   - Go to https://www.weatherapi.com/register.aspx
   - Sign up (free)
   - Copy your API key
3. Open `script.js` in text editor (Notepad/VSCode)
4. Replace `YOUR_API_KEY_HERE` with your real key
5. Save and refresh browser
6. Test:
   - Search "London" or any city
   - Try "xyz" for error
   - Click 📍 My Location (allow location)
   - Reload page - remembers last city

## Features
- Search any city weather (temp °C, condition, humidity, wind, feels like)
- Loading spinner and errors
- Mobile friendly
- Uses localStorage to save last search
- Geolocation button

## API used
WeatherAPI.com - free tier (1,000 calls/day)

## Limitations
- Needs internet
- API key required (free)
- No forecast (current only)
- Simple emojis for weather

Enjoy! 🌤️
